Kento Scroll Jump Top

== Description ==

Kento Scroll. Jump. Top. Is a simple and very light plugin that will take your visitors to the top of the page once they are scrolling through your page.


Plugin Features

+ Select from 20+ icons
+ Smooth transition
+ Easy install and active


Plugin Demo and Support: http://kentothemes.com/items/plugins/kento-scroll-jump-top/

== Installation ==

1. Download from WordPress Plugin Marketplace or download the zip file
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. The plugin will automatically ENABLE itself once its activated.
4. You will see a tab on the left which says Scroll To Top. You can select icons from settings.
